# Swagger
Contains Pipes Framework API documentation.

## Usage
- Change `.dockerignore` file
- Change `config.json` file
- Change `Dockerfile` file
- Run `make build TAG=master` command
- Run `docker run -p 8080:8080 dkr.hanaboso.net/pipes/pipes/swagger:master` command
- Open `http://127.0.0.1:8080` and evaluate results
