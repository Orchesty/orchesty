## Auth

Authentication flow is described on this [wiki page](https://hanaboso.atlassian.net/wiki/spaces/PIP/pages/3228958721/Orchesty+authorization).

The idea is to use a JWE token that is pregenerated by a root app
(tenant's external app). This token is passed to Applinth backend
which return a different (access) token, that is than used for
authorization, until it expires.

For local environment, there is a script for generating such a token.

Run a browser in background and open a locally started frontend
application with JWE token passed in a url query parameter:

```
( firefox "http://localhost:8080/?u=$(node JWEGen.js)" &>/dev/null & )
```
