PIPES offers a library of applications and connectors, which are ready to be used in your processes. We can freely extend this set and create custom collections of new elements. We can see the list of prepared applications further on this page. 

## Appstore installation
Appstore is distributed as a public bundle, more precisely **Symfony bundle**. Installation is possible within every service that implements **PHP-SDK** PIPES. Applications are ready to authorize in the interface [PIPES Admin](/docs/cs/admin) after installation and their connectors are available in [Process editor](/docs/cs/process-editor).

More information in this tutorial [How to use connectors from PIPES Appstore collection](/docs/cs/tutorials/appstore).
