# Applications dashboard

The Applications tab shows connector request activity grouped by application.

## Heatmaps

Each installed application gets its own heatmap. Rows represent individual connectors within the application, columns are time buckets.

- **Green cells** = successful requests (darker = more requests).
- **Red cells** = at least one failed request in that time slot.

Above each heatmap you can see the **total request count** and the **failed count** for the selected time range.

Click a cell to open the connector detail for that specific connector and time slot.
