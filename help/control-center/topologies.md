# Topologies dashboard

The Topologies tab shows aggregated run statistics for all topologies.

## Table columns

| Column | Description |
|--------|-------------|
| **Topology** | Topology name with version. Links to the topology detail page. |
| **Processes run** | Total number of processes executed in the selected time range. |
| **Failed processes** | Number of failed processes. Shown as a red badge when above zero. |
| **Last run time** | Timestamp of the most recent process execution. |
| **Last run status** | Status of the last run: green (success), blue (running), red (failed). |

## Filters

Use the quick filter to show **All**, **Success**, **Running**, or **Failed** topologies.

## Row actions

- **Search icon** -- navigates to the topology detail page.
- **Funnel icon** -- switches to the Processes tab filtered to this topology.
